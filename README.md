# DailyCommit
Commmmmmmit every day!

Telegram version of [geekhub-lab/commit-alarm](https://github.com/geekhub-lab/commit-alarm)

## How to run this chat bot?

  1. `pip install -r requirements.txt`
  2. `./setup.sh`
  3. `python index.py`
